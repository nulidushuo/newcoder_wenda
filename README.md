## 二、项目环境搭建
* **操作系统** ： Windows 10
* **IDE** : IntelliJ IDEA 2019专业版
* **框架** ： Spring Boot 2.1.6
* **依赖管理工具** ： Maven
* **依赖项** ： Web, markerfree(原velocity已经不支持）, AOP, DevTools
* **版本控制** ： git
